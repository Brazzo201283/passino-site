# passino-site

Die Website zu **Passino** — Landing, Datenschutzerklärung, Impressum.

- Live: https://passino.app (GitHub Pages, Custom Domain)
- Bewusst statisch: kein Framework, kein Tracking, keine externen Ressourcen (auch keine Google Fonts — DSGVO).
- Datenschutzerklärung: Quelle der Wahrheit für die Store-Einreichung (siehe App-Repo Issue #25).

## Struktur
- `index.html` — Landing
- `datenschutz/index.html` — Datenschutzerklärung (Stand Juli 2026)
- `impressum/index.html` — Impressum (§ 5 DDG)
- `style.css` — gemeinsames Stylesheet (Light + Dark)
- `CNAME` — Custom Domain für GitHub Pages

## Vor Veröffentlichung ausfüllen
Gelb markierte Platzhalter in Datenschutz + Impressum: Name, Anschrift, Kontakt-E-Mail.
